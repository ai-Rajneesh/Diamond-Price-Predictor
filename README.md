## End-to-End ML Project: Diamond-Price-Prediction

### 1.Create new Environment
```
conda create -p env python==3.8
```

### 2.Activate your environment
- For bash terminal
```
source activate env/
```
- For CMD prompt
```
conda activate env/
```

### 3.Install your requriments file

```
pip install -r requirements.txt
```
